About This Project

This project is a creative To-Do List Web App made using HTML, CSS, and Bootstrap. No JavaScript is used, making it fully static yet visually interactive.

Clean and Attractive Design: The app features gradient backgrounds, colorful cards, and hover effects, giving it a modern look.

Styled Sample Tasks: Tasks have checkboxes and subtle hover animations to simulate real interactivity.

Responsive Layout: Works on desktops, laptops, tablets, and mobile devices. The layout adjusts automatically to any screen size.

Focus on User Interface: Unlike basic To-Do apps, this project emphasizes design and usability, making it stand out from standard projects.

Future Ready: Although tasks cannot be saved or added yet, the layout is prepared for adding JavaScript or backend functionality later.
